### Machine Learning Foundations & Evaluation 
*Focus: Building reliable models and measuring success with precision.*

* **Diagnostic Modeling:** Created a classification pipeline for Breast Cancer detection, prioritizing high recall to minimize missed diagnoses.
* **Safety-Critical Analysis:** Developed a toxicity prediction model for biological data (Mushroom dataset) using advanced classification techniques.
* **Metric-Driven Optimization:** Moved beyond "Accuracy" to evaluate models using ROC Curves, AUC, and F1-Scores to ensure performance in imbalanced datasets.
* **Algorithm Breadth:** Implemented both Regression (for continuous values) and Classification (for categorical labels) to solve diverse data challenges.
